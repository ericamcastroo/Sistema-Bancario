## :computer: Rodando o Projeto

```shell
# 1. Clone o projeto

git clone <urlProjeto>

# 2. Instale as dependências

npm install

# 3. Execute o backend

npm run backend

# 4. Execute o Frontend

npm run dev
```

## :sassy_man: Endpoints

- GET /user - Lista todos os usuários
- GET /user/:id - Busca o usuário do id indicado
- POST /user- Cria um usuário
